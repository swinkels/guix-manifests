This directory contains the files to recreate the Python development container
for one of my customers.

First, install the correct Guix version[fn:1]:
#+begin_src sh
GUIX_EXTRA=$HOME/.guix-extra

mkdir -p "$GUIX_EXTRA"/orderex
guix pull --channels=channel-specs.scm --profile="$GUIX_EXTRA/orderex/guix"
#+end_src

#+RESULTS:

This only has to be done once.

Bash script ~./run-container.sh~ creates and starts the Python development
container.

The following commands, which need to be executed /in a running container/,
install uv, multiple Python versions and several Python packages:
#+begin_src sh :noeval
curl -LsSf https://astral.sh/uv/0.9.2/install.sh | sh

uv python install 3.13
uv python install 3.14

uv tool install nox@2025.05.1 --python 3.13
uv tool install python-lsp-server@1.13.1 --python 3.13
#+end_src
This only has to be done once.

* Footnotes

[fn:1] For details, see the section on [[https://guix.gnu.org/cookbook/en/html_node/Reproducible-profiles.html][reproducible profiles]] in the Guix
cookbook.
